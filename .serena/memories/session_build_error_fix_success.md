## 🎯 問題回顧
- **錯誤訊息**：`[plugin vite:vue] chapters/08-advanced-techniques.md (1035:52): Duplicate attribute.`
- **根本原因**：VitePress的Vue解析器與Python Jinja2模板語法衝突
- **影響範圍**：第8章智能報告生成器中的所有模板語法

## 🔧 修復方案
使用HTML實體編碼將所有Jinja2語法轉換：
- `{{` → `&#123;&#123;`
- `}}` → `&#125;&#125;`
- `{%` → `&#123;%`
- `%}` → `%&#125;`
