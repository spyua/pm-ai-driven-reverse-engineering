歡迎來到AI驅動逆向工程教學指南！這裡將引導您從零開始，掌握使用AI工具進行程式碼分析的完整技能。

## 🎯 學習目標

通過本教學系統，您將能夠：

- **建立AI驅動的程式碼分析思維** - 學會如何有效運用AI工具理解複雜程式碼
- **掌握Cursor編輯器的進階功能** - 深度使用AI輔助的程式碼編輯和分析能力
- **學會系統性分析既有程式碼** - 建立結構化的程式碼審查和理解方法
- **培養產品偵探的洞察能力** - 從程式碼反推產品需求和業務邏輯
- **建立高效的專案逆向工程流程** - 標準化的分析流程和最佳實務

## 🤖 Claude Agent 開發流程

### 必讀文檔
開始章節開發前，請仔細閱讀：
- 📖 **[開發指南](../../DEVELOPMENT_GUIDE.md)** - 完整的開發流程和標準
- 📋 **[檢查點機制](../checkpoints/)** - 品質控制和審核標準
- 🛠️ **[資源工具](../resources/)** - 可用的開發資源總覽

### 開發順序
**嚴格按照以下順序進行**：
1. 閱讀對應章節的規劃指南
2. 研究 tutorial-sample-project 範例專案
3. 整合 tutorial-material 教學素材
4. 運用 reverse-system-prompt 系統提示
5. 撰寫符合標準結構的章節內容
6. 完成 checkpoint 審核

### 標準章節結構
每個章節必須包含：
- 📋 學習目標 (可驗證的具體目標)
- 🎯 前置知識 (先備條件和相關章節)
- 💡 核心概念 (理論基礎 20%)
- 🛠️ 實際操作 (操作步驟 50%)
- 🎮 實戰演練 (練習案例 20%)
- ✅ 總結與檢查 (學習驗證 10%)
