# AI驅動逆向工程教學計畫指南

## 專案概述

本教學計畫旨在教導產品經理和開發者如何運用AI工具（特別是Cursor編輯器）進行有效的程式碼逆向工程和系統分析。透過9個循序漸進的章節，學習者將掌握從基礎設定到進階分析的完整技能。

## 教學目標

- 建立AI驅動的程式碼分析思維
- 掌握Cursor編輯器的進階功能
- 學會系統性分析既有程式碼
- 培養產品偵探的洞察能力
- 建立高效的專案逆向工程流程
